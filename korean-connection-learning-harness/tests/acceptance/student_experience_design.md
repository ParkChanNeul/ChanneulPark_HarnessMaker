# Acceptance Scenario: Student Experience Design

## Mode

`render_materials`

## Input

A valid `lesson_blueprint`, `practice_plan`, and `student_deck_spec`.

## Must Pass

- Student deck mode is the default material route.
- Slide titles are learner-facing.
- Internal headings such as `Cultural Tension`, `Social Meaning`, and `Grammar Tool` are absent from visible titles.
- Design manifest maps materials to source contracts.
- Accessibility notes are present.
