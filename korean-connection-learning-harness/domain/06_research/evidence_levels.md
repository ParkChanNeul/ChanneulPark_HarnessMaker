# Evidence Levels

## Purpose

Labels the strength of evidence behind a teaching rule or domain update.

## Levels

```yaml
level_0:
  name: "hypothesis"
  meaning: "plausible idea with no direct learner evidence yet"
level_1:
  name: "single observation"
  meaning: "seen once in a learner interaction or lesson result"
level_2:
  name: "repeated observation"
  meaning: "seen across multiple lessons or learners"
level_3:
  name: "supported practice"
  meaning: "consistent with domain references and repeated classroom evidence"
level_4:
  name: "approved standard"
  meaning: "human-approved rule in domain or contract files"
```

## Usage Rule

Agents must state the evidence level for any research insight, new progression rule, or proposed change to default lesson behavior.
