# Sequence Rules

## Sequencing Inputs

- learner_context_snapshot
- grammar mastery gaps
- conversation skill gaps
- mission history
- real exposure needs
- review schedule

## Rule

Do not choose the next lesson because the culture topic is interesting. Choose it because it advances a learner state, reviews targets, and creates a realistic transfer opportunity.

## Default Beginner Pattern

1. Stabilize short polite survival moves.
2. Move into repeated service flow.
3. Add reactions and ask-back.
4. Transfer request and politeness tools across new situations.
5. Add identity-specific modules only when learner exposure supports them.
