# Module Map

| module_id | goal | core situations | likely grammar |

| survival_korean | avoid freezing with short polite Korean | taxi, help, directions | -주세요, -요, 잘 모르겠어요 |

## Rule

Modules help route situations. They do not decide mastery or replace learner state.
