# Spiral Review Rules

## Default Review Cadence

- next lesson: immediate retrieval
- after 2 lessons: transfer in a different situation
- after 4 lessons: delayed production without model

## Review Target Selection

Each lesson should include:

- one new target by default
- two to four review targets by default
- at least one retrieval target
- at least one transfer target

If the lesson deviates, include `override_reason` in the progression plan.
