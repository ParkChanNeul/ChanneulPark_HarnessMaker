# Student Experience Rules

Student-facing materials should help the learner act, not admire a theory.

## Required Student Actions

Each major screen or handout section should ask the learner to:

- say
- choose
- repeat
- react
- ask back
- recombine
- role-play
- recall
- prepare a mission

## Visual Rules

- Keep visible text short.
- Separate grammar, meaning, and culture visually.
- Do not expose teacher-side reasoning headings as slide titles.
- Do not hide review targets for visual simplicity.
- Use teacher notes separately from visible student screens.
