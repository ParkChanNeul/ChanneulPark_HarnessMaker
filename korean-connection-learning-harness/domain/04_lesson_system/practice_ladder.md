# Practice Ladder

```text
Recognition
-> Controlled Practice
-> Guided Production
-> Independent Production
-> Transfer
-> Delayed Retrieval
```

## Practice Types

- substitution drill
- question and answer drill
- ask-back drill
- reaction timing
- sentence recombination
- role play
- error repair
- real-world mission
- delayed recall

## Rule

The practice plan must make mastery evidence collectable. A fun activity without evidence is not enough.
