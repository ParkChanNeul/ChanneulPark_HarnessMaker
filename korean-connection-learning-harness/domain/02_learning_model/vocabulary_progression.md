# Vocabulary Progression

Vocabulary progression supports grammar and conversation targets. It is not a memorization dump.

## Categories

- foundation: verbs, adjectives, particles, numbers, counters, time, location
- situation: cafe, shopping, clinic, PT, creator comments, appointments
- function: request, refusal, clarification, reaction, ask-back, confirmation
- repair: "I do not know", "say it again", "slowly", "one moment"

## Review Rules

- New vocabulary should be limited to what the learner needs for the mission and drills.
- Each weekly pack should include a small review set and a Q/A drill set.
- Vocabulary is stronger when tied to a grammar target and real response pattern.
- Do not mark vocabulary stable from passive recognition only.
