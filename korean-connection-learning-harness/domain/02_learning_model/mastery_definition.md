# Mastery Definition

## Mastery Evidence Ladder

```text
recognition
-> controlled practice
-> guided production
-> independent production
-> transfer to new situation
-> delayed retrieval
```

## Evidence Types

- recognition: student identifies meaning or function.
- controlled: student fills or substitutes with a clear model.
- guided: student produces with prompts or limited choices.
- independent: student produces without model in role play.
- transfer: student uses the target in a different situation.
- delayed: student retrieves it in a later lesson or follow-up.

## Decision Rule

Do not mark a grammar, expression, or conversation skill as stable unless evidence exists for independent production and at least one later retrieval or transfer checkpoint.
