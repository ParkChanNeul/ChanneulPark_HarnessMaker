# Conversation Skill Progression

Conversation skills are tracked separately from grammar.

## Skill Areas

- answer_question
- ask_question
- ask_back
- reaction_before_answer
- repair_when_confused
- soft_refusal
- confirm_next_step
- close_conversation
- adjust_register

## State Scale

```text
not_introduced -> introduced -> guided -> developing -> stable -> transfer_needed
```

## Evidence

Each skill needs evidence from role play, independent production, real mission, or next-lesson check. Teacher impression alone is not enough.
