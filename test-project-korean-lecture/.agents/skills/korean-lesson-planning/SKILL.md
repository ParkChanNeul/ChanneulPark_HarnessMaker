# Korean Lesson Planning

한국어 수업 요청을 수업 설계안으로 바꾸는 스킬이다. lesson_planner가 사용한다.

## 입력

- 사용자 요청 원문
- 학습자 수준, 수업 시간, 수업 목적
- 이전 `_workspace/` 산출물

## 절차

1. 학습자 수준을 A1/A2/B1/B2 중 하나로 가정하거나 명시한다.
2. 수업 목표를 "학생은 수업 후 X를 할 수 있다" 형태로 1-2개 작성한다.
3. 핵심 문법 1개, 핵심 표현 3-5개, 필수 어휘 6-10개를 고른다.
4. 60분 기준 수업 흐름을 작성한다.
5. 활동 설계자가 이어받을 제약과 검토자가 확인할 리스크를 표시한다.

## 출력 형식

```markdown
# 한국어 수업 설계안

## 요청 요약
## 가정
## 학습 목표
## 핵심 표현
## 수업 흐름
| 단계 | 시간 | 목표 | 교사 행동 | 학생 행동 |
## 활동 설계 제약
## 검토 리스크
```

## 품질 기준

- 수업 목표와 활동이 직접 연결되어야 한다.
- 한 수업에 새 문법을 너무 많이 넣지 않는다.
- 말하기 수업이면 학생 발화 시간이 충분해야 한다.
