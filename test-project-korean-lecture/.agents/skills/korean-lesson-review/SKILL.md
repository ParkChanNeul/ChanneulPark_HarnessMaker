# Korean Lesson Review

한국어 강의 산출물을 검수하는 스킬이다. lesson_reviewer가 사용한다.

## 입력

- `_workspace/01_lesson_plan.md`
- `_workspace/02_activity_pack.md`
- 사용자 요청 원문

## 절차

1. 수업 목표와 활동이 같은 방향인지 비교한다.
2. 수준 대비 어려운 문법, 어휘, 활동을 찾는다.
3. 수업 시간이 45-60분 안에 가능한지 확인한다.
4. 한국어 예문이 자연스럽고 수업 상황에 맞는지 검토한다.
5. 수정 요청을 high/medium/low로 구분한다.

## 출력 형식

```markdown
# 한국어 수업 검수 리포트

## 판정
통과 / 수정 필요 / 미검증

## 주요 발견
| 심각도 | 위치 | 문제 | 수정 제안 |

## 목표-활동 정렬
## 시간 배분
## 한국어 자연성
## 최종 권장 수정
```

## 품질 기준

- 단순 존재 확인보다 목표와 활동의 연결을 검토한다.
- 수정 제안은 parent가 바로 반영할 수 있게 구체적으로 쓴다.
