# Korean Activity Design

한국어 수업 설계안을 실제 수업 자료로 바꾸는 스킬이다. activity_designer가 사용한다.

## 입력

- `_workspace/01_lesson_plan.md`
- 사용자 요청 원문
- 수업 시간과 학습자 수준

## 절차

1. 목표 표현마다 짧은 controlled practice를 만든다.
2. 학생이 직접 말해야 하는 free practice를 1개 이상 만든다.
3. 교사용 진행 문구를 자연스러운 한국어로 작성한다.
4. 학생용 예문과 답변 예시를 제공한다.
5. 10-15분 숙제와 다음 수업 연결 질문을 만든다.

## 출력 형식

```markdown
# 한국어 수업 활동 패키지

## 교사용 진행안
## 학생용 핵심 표현
## 연습 1: 형태 연습
## 연습 2: 짝활동
## 연습 3: 역할극 또는 말하기 확장
## 숙제
## 다음 수업 연결 질문
```

## 품질 기준

- 모든 활동은 학습 목표 중 하나와 연결되어야 한다.
- 지시문은 수업 중 바로 읽을 수 있게 짧고 명확해야 한다.
- 숙제는 너무 길지 않아야 한다.
